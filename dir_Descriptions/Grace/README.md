# Graceの呼び方

*callSisters*を以下のように記載すれば呼び出せますよ！
```
void callSisters( void ){

     Grace = callGrace() ;

     return ;
}
```

# Graceができること
## fnService()
```
Grace.fnService(
     portNumber
) ;
```
指定する portNumber で HTTP Webサーバーを起動できます！  
Sisters/dir_Room/Grace/ に設置した ghtml ファイルをサービスします！  
ghtml は Grace が読む html ファイルです！
